Docker commande

1. Pour vérifie que Docker est installé et tourne:
```
docker --version
```
-Et pour vérifier si Docker tourne :
```
sudo systemctl status docker
```
-Pour le demarer
```
sudo systemctl start docker

```
-Voir les conteneurs en cours d’exécution
```
docker ps
```
-. Voir les images Docker
```
docker images
```
